WASM UEFIPatch

UEFI firmware patcher

Uses UEFIPatch under the hood.

Instructions:
1. Load your ROM file.
2. Configure patches.
3. Click "Patch".

Notes:
Wasm module is ~3MiB so it may take a while to load on slow connections.
Old UEFITool engine is used, it has been known to render some BIOSes unbootable. Proceed only if you have recovery method (or don't care).


Patches:

CSB - Circumvent Secure Boot. Replaces all EFI_SECURITY_VIOLATION with EFI_SUCCESS in SecurityStubDxe. Only known working on AMD AMI boards.
custom - Custom patches. Enter them below. Patch Format Guide
